## 关键原则（贯穿所有步骤）

1. **增量可运行**：每个步骤完成后都应能独立运行和验证
2. **兼容优先**：重构 demo 代码时，保持原有功能可用
3. **Schema 驱动**：所有数据以 Schema v2.0 为唯一标准
4. **审核闭环**：生成→审核→回改→复审 四步，不跳过
5. **状态可恢复**：所有 Agent 通过 `.agent-state.json` 支持中断续跑
6. **Provider 无关**：LLM / 图片生成 / 视频生成 通过抽象层接入，不绑定特定厂商
